Boarding Data Analysis of a Public Transit

Purpose of Analyzing Transit Ridership Data

The boarding alighting data collected via TransitWand contains spatial data. Analyzing this data provides valuable insights about the transit system and enables its performance evaluation. Some key elements that can be analyzed are:

  • Passenger activity,
  • Spatial pattern recognition (Recognizing areas with high or low usage),
  • Discerning passenger demands,
  • Stoppage/ station efficiency,
  • Differentiate between Peak vs Off-peak performance,
  • Service gaps,
  • Revenue estimation,
  • Load Factor,
  • Transfer patterns,
  • Poentital areas for improvement etc.

    • Preparing Raw Data

    The collected data came in Shapefiles and csv files. Which I then processed and visualized for the purpose analysis.

    The ride occupancy, travel time, operational speed and estimated revenue was generated with the data. Revenue can be estimated through Revenue Passenger Kilometer (RPK) otherwise known as the Passenger Load Unit. Load Factor for each trip was also determined. These help us to understand and analyze the trends in passenger usage. It also allows us to compare between different routes and set a performance benchmark.


    Data Visualization

    Since we don’t have properly utilized bus stops, individual stoppage performances are hard to assess. So, the surveyed route has been split into multiple segments. The demands for each segment are then visualized.



    Visualizing Boarding-Alighting with Pie Charts


    Heatmaps

    Heatmaps also depict activity patterns. These were generated separately for each trip. Higher passenger counts are shown in the heatmap as darker and denser patches.


    Occupancy Curve

    I also prepared Occupancy curves for all 4 individual trips. Occupancy curves are a great way to see how crowded or empty the vehicles run. In conjunction with the Passenger Load Factor and RPK (Revenue Passenger Kilometer) it gives a great insight on the performance of a transit line.

    Outcome

    Proper visualization provides valuable insights. Analyzing spatial data would reveal inefficiencies in the transit system that would otherwise go unnoticed. It is important for Transport Planners to carefully prepare the data.

    I am optimistic that the data visualizations I did were of great value for the department. Utilization of such data have great potential to drastically increase the efficacy of Dhaka's transit system.
